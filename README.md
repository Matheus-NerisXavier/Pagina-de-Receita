# 🍰 Página de Receita - Cupcake de Café com Chantilly

Este é um projeto simples de uma página de receita, criado como parte dos meus estudos iniciais em **HTML e CSS** pela [Rocketseat](https://www.rocketseat.com.br/). A receita escolhida foi um delicioso **Cupcake de Café com Chantilly**, perfeita para acompanhar um café!
